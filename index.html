Excel Schedule Maker Guide

Learn how to use the scheduling workbook step by step. Use search, contents, or jump directly into any section.

Important: Confirm that you are using the latest version of the Schedule Maker before following this guide. The current version is 1.1.0, available as of March 16, 2026. Earlier versions may not support all features described in this guide.

When you first open the Schedule Maker Excel file, make sure to enable content. After doing so, you will be prompted to choose either Getting Started or Import Prior Data.

See the details of both options below and click the button that applies to you.

Getting Started

Select Getting Started if you are:

  • Using the Schedule Maker for the first time.
  • Setting up a new district or team.
  • Creating a schedule from scratch.

This option will allow you to enter district information, add locations, add employees, define the district manager, and configure sheet settings.

This option is the best choice if you do not have an existing schedule file to pull data from.

For guidance through this process, jump into the Getting Started section.

Import Prior Data

Select Import Prior Data if you are:

  • Updating to a new version of the Schedule Maker.
  • Continuing work from a previous file.
  • Looking to reuse existing data.
  • Inheriting stores or locations from another manager's schedule file.

This option allows you to import locations and employees, import default and active schedules, and optionally import settings and district manager information.

This option is the recommended option if ou have already been using the Schedule Maker and want to avoid re-entering information.

For guidance through this process, jump into the Import Prior Data section.

This section will guide you through the starting steps to get the schedule maker up and running for your district's schedule.

When you first open the Schedule Maker Excel file, make sure to enable content. After doing so, you will be prompted to choose either Getting Started or Import Prior Data. Click Getting Started to continue.

You will need ample time to successfully complete this process. Quiting in the middle of this process will require you to complete it manually later.

After selecting Getting Started you will be presented with the main setup menu. This menu is designed to be intuitive and will guide you through the required initial information for your district.

In this menu, you must enter the following information in the corresponding fields:

  • District Number
  • District Manager's Name
  • Number of Locations in the District
  • Number of Employees in the District

Please ensure this information is entered correctly before continuing. If any details are entered incorrectly, they can be edited later after the Getting Started process is complete.

Once all required fields are filled out, select Submit to continue.

Entering a Date

After clicking Submit, you will be asked to enter a date of the Week. The schedule will automatically set the dates for the week based on the date you enter.

For accuracy, the date must be formatted as '01/01/2000'.

The date for the week can always be edited later.

Adding Locations

Next, you will be guided through a series of menus to add your district's locations one at a time. Each location will be added individually to ensure accuracy.

For detailed instructions, see the Adding Locations section of this guide.

Adding Employees

After all locations have been added, you will be prompted to add employees, again one at a time. This step ensures that each employee is properly added and can have a default schedule made using the newly added locations.

For detailed instructions, see the Adding Employees section of this guide.

District Manager's Schedule

After all locations and employees have been added, the final screen will display the District Manager Schedule window.

Here, you will set the District Manager's schedule to complete the initial setup process.

For detailed instructions, see the Editing the District Manager Schedule section of this guide.

Preferred Settings

After completing the District Manager's schedule, you will be prompted to set your preferred settings.

A settings menu will appear, allowing you to configure how the Schedule Maker behaves by default. The following options are available:

  • Default PTO Hours Amount
  • District Number
  • Update DM Schedule w/ Export
    • - (On or Off — Off by default)
  • Reset Warnings With Date
    • - (On or Off — On by default)
  • Autopopulate Schedule Editor
    • - (On or Off — On by default)
  • Value for Autopopulate
    • Current Schedule (default)
    • Employee Defaults
    • Weekday Defaults
  • Schedule Color Theme

Each of these settings controls default behaviors and visual preferences within the Schedule Maker.

Full explanations for each option can be found in the Settings section of this guide.

Saving Settings

After selecting your preferred options, click the Save button.

You will be prompted to confirm your selection, as saving will overwrite any existing settings. Select Yes to apply and save the new settings.

Side Note: Clicking the Close button will close the menu without changing any of the default settings.

Completion

Once settings are saved, the Getting Started process is complete and the Schedule Maker is fully ready for use.

This section covers how to add locations after you have completed the Getting Started process.

Opening the Add Location Menu

  1. In the top-left corner of the Excel sheet, click the Additional Options button.
  2. An options menu will appear.
  3. Select Add New Location.

Entering Location Information

After selecting Add New Location, a location setup menu will be displayed.

In this menu, you will enter the following information:

  • Location Name
  • Permitted Hours per Week
  • Days the Location is Open
  • Number of Open Hours per Day

By default, the menu will typically be pre-filled with Atlantic Self Storage's standard operating days and hours. These values can be adjusted if the location has unique operating hours.

Important Rules and Validation

Please note the following requirements when entering location information:

  • Location names must be unique and cannot match the name of any existing location or employee.
  • Special characters are not allowed in location names.
  • Permitted hours per week must exceed the total number of open hours for the location each week.

If any of these conditions are not met, the location will not be saved.

Saving the Location

Once all required information has been entered and verified, click the Save Location button to finish adding the location.

The location will then be available for scheduling and employee assignment.

Side Note: Clicking the Close button will close the menu without saving any filled out information. This will also skip adding one location if you are in the Getting Started process.

Importing Locations

An alternative method of adding locations that is available is importing them.

Please refer to the Importing Schedule Data section for detailed instructions for doing so.

This section covers how to add employees after you have completed the Getting Started process.

Opening the Add Employee Menu

  1. In the top-left corner of the Excel sheet, click the Additional Options button.
  2. An options menu will appear.

Entering Employee Information

After selecting Add New Employee, an employee setup menu will be displayed.

In this menu, you will enter the following information:

  • Employee Name
  • Permitted Hours per Week
  • Employee Note (optional)
  • Salaried Status

The Employee Note, if provided, will appear directly under the employee's name on the schedule. This can be used for role titles, availability notes, or other helpful identifiers.

Important Rules and Validation

Please note the following requirements when entering employee information:

  • Employee names must be unique and cannot match the name of any existing location or employee.
  • Special characters are not allowed in employee names or employee notes.

If any of these conditions are not met, the employee will not be saved.

Saving the Employee

Once all required information has been entered and verified, click the Add Employee button to finish adding the employee.

The employee will then be available for scheduling and location assignment.

Side Note: Clicking the Close button will close the menu without saving any filled out information. This will also skip adding one employee if you are in the Getting Started process.

Creating an Employee's Default Schedule

After an employee is successfully added, and at least one location has been added previously, you will be prompted to choose whether you would like to create the employee's default schedule.

  • If you select Yes, you will be taken to the default schedule setup.
  • Default schedules can only be created using locations that already exist in the system.
    • Side Note: This is why locations are added before employees during the Getting Started process.
  • If you select No, the employee will be added without a default schedule.

An employee's default schedule can always be created or edited later, so it is not required to complete this step immediately.

For detailed instructions, see the Creating or Editing an Employee's Default Schedule section of this guide.

An alternative method of adding employees that is available is importing them.

Please refer to the Importing Schedule Data section for detailed instructions for doing so.

This section covers how to update the District Manager's schedule after you have completed the Getting Started process.

Opening the District Manager Schedule Editor

  1. In the top-left corner of the Excel sheet, click the Additional Options button.
  2. An options menu will appear.
  3. Select Edit District Manager Schedule.

District Manager Editor Overview

The District Manager Schedule menu will open and automatically populate with the current District Manager schedule.

If this is being completed during the Getting Started process, the schedule may appear blank.

At the top of the menu is a field labeled District Manager's Name. Enter or update the District Manager's name in this text box.

Below the name field, you will see a text box for each day of the week, laid out horizontally to resemble a calendar week.

Entering the Schedule

Each day's text box is intended to contain the District Manager's schedule for that day. These boxes support multiple lines of text, allowing the schedule to be entered manually in a clear, readable format.

The current standard formatting is:

  • Enter "ON" for each day the District Manager is working.
  • For Saturday and Sunday, enter the names of the managers working those days.
  • Any location or meeting the District Manager is scheduled for may also be included if needed.

This standard format provides flexibility while keeping the schedule easy to read.

Saving the District Manager's Schedule

Once all District Manager schedule information has been entered and verified, click the Save Schedule button.

You will be prompted to confirm your changes, as saving will overwrite the existing District Manager schedule. Select Yes to confirm and save the updated schedule.

Side Note: Clicking the Close button will close the menu without saving any filled out information. This will leave the District Manager's schedule blank if you are in the Getting Started process.

Creating an employee's default schedule is highly recommended, as it makes future schedule editing faster and more efficient. When editing schedules later, you can use a single button to autofill the schedule editor with the selected employee's default schedule.

Opening the Default Schedule Editor

  1. In the top-left corner of the Excel sheet, click the Additional Options button.
  2. An options menu will appear.
  3. Select Edit Employee Default Schedule.
  4. Choose the desired employee from the drop-down menu.
  5. Click the Edit button.

Purpose of Employee Default Schedules

Employee default schedules define an employee's typical weekly work pattern and are used to speed up schedule creation and editing.

When working in the schedule editor, you can use a single button to autofill the entire week with an employee's default schedule instead of entering each day manually. This helps ensure consistency and reduces repetitive data entry.

Employee default schedules are especially useful for:

  • Employees with consistent weekly hours.
  • Reducing scheduling errors.
  • Quickly rebuilding schedules week over week.

Default schedules can be updated at any time, and changes will apply the next time the autofill feature is used.

Default Schedule Behavior

By default, an employee's schedule is set to off for every day of the week until changes are made.

The default schedule editor is organized using tabs for each day of the week, making it easy to configure one day at a time.

Day Off Configuration Options

Within each day's tab the following options are available:

  • Off Day Checkbox
    • When checked, the employee is marked as off for that day.
  • PTO Checkbox
    • Becomes available when the employee is marked as off.
    • This indicates the day is a PTO day. The employee will be off but will have the default PTO amount added to their total working hours for the week.

Using Weekday Defaults

To the right of the off checkboxes is a button with a circular arrow icon, which will autofill the open day tab with the weekday default.

For more weekday default information, see the Editing the Weekday Default Schedules section of this guide.

Work Details

Below the checkboxes, enter the employee's common work details for that day:

  • Location (drop-down selection)
  • Clock In Time
  • Clock Out Time
  • Break (minutes)
  • Note

Second Location Option

Below the primary work details is a checkbox to indicate whether the employee works at a second location that day.

If enabled, additional fields will appear allowing you to select:

  • Second Location (drop-down selection)
  • Second Location Clock In Time
  • Second Location Clock Out Time

Save the Default Schedule

Repeat this process for each day of the week as needed.

Once all desired default schedule information has been entered, click the Save Defaults button to finish.

The employee's default schedule will then be saved and available for autofill when creating or editing schedules.

Side Note: Clicking the Close button will close the menu without changing the employee's default schedule.

Weekday default schedules are used to quickly autofill a single day when editing schedules. These defaults are especially helpful when most employees share similar working hours on certain days of the week.

Opening the Weekday Default Schedule Editor

  1. In the top-left corner of the Excel sheet, click the Additional Options button.
  2. An options menu will appear.
  3. Select Edit Weekday Default Schedules.

Purpose of Weekday Defaults

The weekday default schedule editor functions similarly to the employee default schedule editor, but instead of saving defaults per employee, it saves standard schedules for each day of the week.

These defaults can be applied with a single click when editing schedules.

Example: If most employees typically work 9:30 AM to 6:00 PM on Fridays, you would set Friday's weekday default to those hours. When editing an employee's schedule, you can then autofill just Friday using the preset.

Default Weekday Settings

By default, the weekday schedules are set as follows:

  • Monday - Saturday:
    • Clock In: 9:30 AM
    • Clock Out: 6:00 PM
    • No Break
    • Note: Lone Trade
    • No Second Location
  • Sunday:
    • Set as Off

These defaults can be customized as needed to match your district's scheduling patterns.

Editing Weekday Defaults

The weekday default editor uses the same layout and options, except for location options, as the Employee Default Schedule Editor, including:

  • Day-Specific Tabs
  • Off and PTO Options
  • Clock In and Clock Out Times
  • Break Duration
  • Notes
  • Second Location Options

Make any changes needed for each day of the week to reflect your standard scheduling practices.

Saving Weekday Defaults

Once your weekday defaults are configured, click the Save Defaults button to apply your changes.

These weekday defaults will now be available for quick autofill of individual days when creating or editing schedules.

Side Note: Clicking the Close button will close the menu without changing the weekday default schedules.

The Settings menu allows you to control the default behaviors, warnings, automation, and visual preferences within the Schedule Maker. Each setting can be updated at any time and will affect how schedules are created, edited, and displayed.

Opening the Settings Menu

  1. In the top-left corner of the Excel sheet, click the Additional Options button.
  2. An options menu will appear.
  3. Select Open Settings.

Default PTO Hours Amount

This setting is a numeric value (default: 8) that determines how many hours are added to an employee's weekly worked hours when they are scheduled off for a PTO day.

Important notes:

  • PTO hours count toward the employee's total hours for the week.
  • PTO hours do not contribute to any location's total hours.
  • If an employee uses less than the default amount of PTO (for example, 4 hours instead of 8):
    1. Temporarily change this setting to the desired value.
    2. Schedule the employee off for PTO on that day.
    3. Change the setting back to its default value if desired.

District Number

This setting is a numeric value that determines the district number displayed in the schedule title.

This value is normally set during the Getting Started process, but is included here in case that step was skipped or needs to be updated later.

Update DM Schedule w/ Export

(On or Off — Off by default)

When enabled, you will be prompted to update the District Manager's Schedule every time you export a schedule.

This can be helpful if:

  • You frequently forget to update the district manager's schedule before exporting.
  • The district manager's schedule often changes week to week.

If disabled, exporting the schedule will not prompt for updates to the district manager's schedule, and you will need to make sure to do so manually before exporting.

Reset Warnings With Date

(On or Off — On by default)

For more details on warnings, see the Warnings section of this guide.

Explaining briefly, warnings appear when:

  • A location is not fully covered on an open day.
  • A location's scheduled hours exceeds its permitted hours.
  • An employee is scheduled for more hours than their permitted hours.

Users may choose to ignore warnings in legitimate scenarios, such as:

  • Approved overtime.
  • Coverage from an employee in another district.
  • Other unique staffing situations.

When this setting is On, all previously ignored warnings will be reset and shown again whenever the schedule week dates are updated. This is based on the assumption that changing dates means a new schedule is being created.

You may consider turning this setting Off if:

  • You expect the same warning to appear every week and want to continue to ignore it.

However, leaving this setting off can be risky, as it may cause new or unintended warnings to be missed.

Autopopulate Schedule Editor

(On or Off — On by default)

When enabled, the schedule editor will automatically populate when opened using the option selected in the next setting.

When disabled, the schedule editor will open completely blank, requiring all information to be entered manually.

Value for Autopopulate

(Current Schedule, Employee Defaults, Weekday Defaults — Current Schedule by default)

This setting determines what content is loaded into the schedule editor when autopopulation is enabled:

  • Current Schedule — Loads in the existing schedule.
  • Employee Defaults — Loads in each employee's default schedule.
  • Weekday Defaults — Loads in the saved weekday default schedules.

These options are named intuitively to describe exactly what will be populated automatically.

Schedule Color Theme

This setting allows you to choose the visual color theme used for the schedule.

Multiple themes are available, including several holiday themed options, allowing you to customize the look of the schedule to your preference or to match the season.

Side Note: Feel free to recommend a custom theme that is not available.

Saving Settings

After selecting your preferred options, click the Save button.

You will be prompted to confirm your selection, as saving will overwrite any existing settings. Select Yes to apply and save the new settings.

Side Note: Clicking the Close button will close the menu without changing any of the active settings.

Warnings are automatically checked any time the schedule is updated, ensuring potential issues are identified as changes are made.

Warnings are informational and are meant to bring attention to possible scheduling oversights.

When Warnings Are Generated

The Schedule Maker checks for the following warning conditions:

  • Location Permitted Hours Exceeded
    • A warning is generated if a location is scheduled for more total hours than its permitted weekly hours.
  • Employee Permitted Hours Exceeded
    • A warning is generated if an employee is scheduled for more hours than they are permitted to work in a week.
  • Location Not Fully Covered
    • A warning is generated if a location is not fully covered for the number of hours it is open on any day of the week.
    • Important Note: A separate warning will appear for each affected day, if applicable.

Viewing Warnings

Warnings do not automatically appear on screen.

In the top left corner of the Excel sheet, there is a button with a circled X icon. If any warnings exist, this button will turn red.

Clicking this button will open the Warnings menu.

Side Note: If this button is not red, there are either no current warnings or all warnings have been ignored by the user. Despite no warnings, the menu can still be opened.

Warnings Menu

The Warnings menu displays an individual list of all detected warnings.

If the list box is empty, there are either no current warnings or all warnings have been ignored by the user.

Below the warnings list are three buttons:

  • Ignore Warning
    • Allows you to select a specific warning from the list and ignore it. Ignored warnings will no longer appear in the list.
  • View Ignored
    • Switches the menu to view all ignored warnings.
  • Close
    • Closes the warning menu.

Warning Indicator Behavior

The warnings button will remain red as long as there are active warnings.

The button will return to its normal state only when:

  • All warnings have been resolved.

    • Or

  • All existing warnings have been intentionally ignored.
Side Note: Ignored warnings may also reappear depending on the Reset Warnings with Date setting when schedule dates are changed.

Viewing and Managing Ignored Warnings

When View Ignored is selected the list updates to show all ignored warnings.

The available buttons change to:

  • Undo Ignore
    • Restores a selected ignored warning back to the active warnings list.
  • Undo All
    • Resets all ignored warnings at once.
  • Back
    • Returns to the original warnings list.

This section explains how to edit or delete an existing location.

Opening the Location Management Menu

  1. In the top-left corner of the Excel sheet, click the Additional Options button.
  2. An options menu will appear.
  3. Select Edit or Delete Existing Location.

A menu will appear allowing you to manage existing locations.

Selecting a Location

At the top of the menu, select the desired location from the drop-down list.

Below the drop-down menu are two buttons:

  • Edit
  • Delete

Editing a Location

If you click Edit, the same menu used when originally adding the location will appear, prefilled with the current location information.

You may modify:

  • Location name
  • Permitted hours
  • Open days and hours

After making changes, click the Save Location button to save updates.

All updates will immediately reflect in the schedule.

Important Rules and Validation

Please note the following requirements when editing location information:

  • Location names must be unique and cannot match the name of any existing location or employee.
  • Special characters are not allowed in location names.
  • Permitted hours per week must exceed the total number of open hours for the location each week.

If any of these conditions are not met, the location will not be saved.

Deleting a Location

If you click Delete, you will be prompted to confirm your selection.

  • Selecting Yes will permanently delete the selected location.
  • Any employees scheduled to work at that location will automatically be set to Off on those days.

This action cannot be undone unless the location is re-created and schedules are manually restored.

This section covers how to use the schedule editor to create and modify the schedule.

Opening the Schedule Editor

  • In the top-left corner of the Excel sheet, click the Edit Schedule button.

Selecting an Employee

At the top of the schedule editor is a drop-down list where you select the employee whose schedule you would like to edit.

When switching to a different employee:

  • The schedule editor will automatically populate based on your Autopopulate Settings.

Applying an Employee's Default Schedule

To the right of the employee drop-down menu is a button with a circular arrow icon.

  • Clicking this button will load the selected Employee's Default Schedule into the editor.

Important Note: After loading a default schedule, you must still click Save to apply it to the current week and before changing the selected employee

Navigating the Weekday Tabs

Below the employee drop-down menu are tabs for each day of the week.

To define and edit a schedule:

  • Click each tab individually.
  • Configure the employee's schedule for that specific weekday within each tab.

Day Configuration Options

  1. Off & PTO Options
    2.
    • Off Checkbox
      • Marks the employee as off for the open weekday.
    • PTO Checkbox
      • Becomes toggable when the employee is marked off. Marks the off day as a PTO day, using hours.
  2. Weekday Default Button
    3.
    • This button is to the right of the Off Checkboxes and has a circular arrow icon.
    • This button, when clicked, loads the Weekday Default Schedule for the open tab.
      • Example: If the Monday tab is open, clicking this button loads the Monday weekday default schedule only.
  3. Location
    4.
    • This is a drop-down menu that allows you to select the location the employee will be at on the open weekday.
  4. Clock In Time
    5.
    • This will be the time the employee will start their day.
    • Format the time as: hh:mm AM/PM
  5. Clock Out Time
    6.
    • This will be the time the employee will either end their day or leave the first location.
    • Format the time as: hh:mm AM/PM
  6. Break
    7.
    • In this box you will input, in minutes, how long the employee's break should be on the open weekday.
    • Example: If an employee should have an hour-long break, you must enter "60" into the break box.
  7. Note
    8.
    • This is the note that will be under the employee's schedule for the open weekday.
  8. Second Location Checkbox
    9.
    • If you intend to schedule an employee at a second location in one day, check this checkbox.
  9. Second Location
    10.
    • This drop-down list becomes toggable when the Second Location CheckBox is checked.
    • This is a drop-down menu that allows you to select the second location the employee will be at on the open weekday.
  10. Second Clock In Time
    11.
    • This text box becomes editable when the Second Location CheckBox is checked.
    • This will be the time the employee will start their shift at the second location.
    • Format the time as: hh:mm AM/PM
  11. Second Clock Out Time
    12.
    • This text box becomes editable when the Second Location CheckBox is checked.
    • This will be the time the employee will end their shift at the second location.
    • Format the time as: hh:mm AM/PM

Daily and Weekly Hours Totals

Within each day's tab, at the bottom, you will see:

  • The date of that weekday according to the current schedule week.
  • The total hours scheduled for that day.

This allows you to quickly verify hours while building the schedule

Side Note: Due to Excel limitations, the total hours viewed in the menu may not update correctly until you click into a new text box. Clicking Save will use the accurate hours and not the hours that can be seen in the menu.

Version and Control Buttons

On the lower left of the schedule editor is a Version Number field.

You may modify this number to define the version of the schedule you are working on.

To the right of this field are the following buttons:

  • Save
  • Close
  • Update Schedule Dates
    •
    • This button is defined with a document icon.

Side Note: The version number will update live. This means, you do not need to click the Save button to update it.

Updating Schedule Dates

There are two ways to update the schedule dates:

  1. Within the Schedule Editor
    2.

    After opening the schedule editor you can click the Update Schedule Dates button at the bottom right of the menu inside the schedule editor. This button is defined with a document icon.

  2. Outside the Schedule Editor
    3.

    In the top left of the Excel sheet, you can click the Update Schedule Dates button. This button is also defined with a document icon.

Clicking either button will prompt you to enter a date.

  • Enter any date within the week you want to build.
  • The schedule maker will adjust the entire schedule to reflect that week.

Side Note: Changing the schedule dates may trigger warnings to reset depending on your settings.

Saving an Employee's Schedule

After making changes for an employee, you must click Save before switching to another employee.

If you change employees without saving, the schedule you entered for the previous employee will not save.

Workflow Recommendation

For best results, I recommend the following workflow:

  1. Select employee.
  2. Use employee's default schedule or the weekday default schedules.
  3. Adjust each day as needed.
  4. Review total weekly hours.
    • Note: Due to Excel limitations, the total hours viewed in the menu may not update correctly until you click into a new text box. Clicking Save will use the correct hours and not always the hours that can be seen in the menu.
  5. Click Save.
  6. Move to the next employee.
  7. Repeat steps 1 through 6 until finished.

Once the schedule is complete, it is ready to be exported and distributed.

There are two ways to export the schedule:

  1. Quick Save Button
    2.
    • Click the Save button (floppy disk icon) in the top left corner of the Excel sheet.
  2. Export Schedule Button
    3.
    • Open Additional Options.
    • Click the Export Schedule button.

Both options will open your File Explorer, allowing you to choose where the files will be saved.

Side Note: If the current schedule has any active warnings, you will be warned before exporting. If you have the Update DM Schedule w/ Export setting on, you will be updating the district manager's schedule before exporting as well.

Default File Name

By default, the exported file will be named:

District_[District Number]_Storage_Export_Schedule_v[Version Number]_[Monday Date]_to_[Sunday Date]

You may rename the file before saving if desired.

Exporting may take a moment depending on system performance, but you will receive a message box that says "Export complete." once the process finishes.

What Gets Exported

Exporting the schedule creates three separate files:

  1. Word Document (Main Schedule)
    2.
    • The schedule is exported as a Word document.
    • This file can be opened and edited further if needed.
    • This is the file that will be emailed to your district.
  2. PDF — Hours Report
    3.
    • A PDF file is created with the same file name you chose, except the ending will include _Hours_Report.
    • This is an export of the Hours Export sheet in the Schedule Maker Excel file.
    • This report includes each location's worked hours along with each employee's worked hours.
  3. Excel File (Schedule Backup Copy)
    4.
    • A copy of the Schedule Maker Excel file is also exported.
    • It uses the same file name you chose.
    • This file contains the exact schedule you created.
    • This version should be used if you need to:
      1. Make schedule adjustment / changes.
      2. Create a revised version due to call out's or other factors

For detailed instructions on creating a revised schedule version, see the Updating the Schedule / Making New Schedule Versions section of this guide.

Sometimes a schedule must be revised after it has already been exported. This may happen for many reasons such as employee availability changes, coverage adjustments, or other last minute updates.

When revising a schedule, it is recommended to use the previously exported Excel schedule file rather than the original Schedule Maker file.

This prevents conflicts if you have already begun working on the following week's schedule in the main Schedule Maker file.

Using the Exported Schedule

  1. Open the previously exported Excel schedule file.

  3. Update the schedule version.
  4. Highlight schedule changes.
  5. Export the newly updated schedule.

Updating the version number helps clearly indicate which schedule version is the most current when distributing revisions.

Side Note: The version number will update live. This means, you do not need to click the Save button to update it.

Choosing a Highlight Color

The color will default to yellow, but there are three highlight colors available.

Removing Highlights

  • Erase Button

  • Clear All Highlights Button

Exporting the Revised Schedule

The Schedule Maker provides a quick way to review total scheduled hours for both employees and locations.

This information can be viewed in the Hours Export sheet within the Schedule Maker workbook.

Accessing the Hours Export Sheet

At the bottom of the Excel workbook, select the sheet labeled Hours Export.

This sheet contains two tables that summarize the hours currently scheduled.

Location Hours Table

The first table displays location hour totals.

Each row contains the following information:

  • Location Name
  • Permitted Hours
  • Total Scheduled Hours

This table allows you to quickly compare how many hours a location is scheduled for versus how many hours it is permitted to operate.

Employee Hours Table

The first table displays employee hour totals.

Each row contains the following information:

  • Employee Name
  • Permitted Hours
  • Total Scheduled Hours

This table allows you to quickly verify whether employees are within their permitted weekly hours.

Viewing Hours on the Schedule

Employee hours can also be seen directly on the main schedule.

  • Each employee's daily hours worked are shown in parentheses next to their scheduled shift.
  • The employee's total hours for the week are shown in parentheses next to their name.
  • A sum of all employee's daily hours worked are shown in the small table below the District Manager's schedule.

Location specific hours can only be viewed in the Hours Export table.

The Import Schedule Data feature allows you to import locations, employees, schedules, and settings from another Schedule Maker file.

This is useful when:

  • Reusing data from a prior schedule.
  • Transferring data between files.
  • Rebuilding a schedule without starting from scratch.
  • Importing data into an updated version of the Schedule Maker.

Opening the Import Menu

  1. In the top-left corner of the Excel sheet, click the Additional Options button.
  2. An options menu will appear.
  3. Select Import Schedule Data.

    4. This will open the file explorer.

  4. Select another Schedule Maker Excel File.

    5. After selecting a compatible file, the Import Menu will appear.

Import Menu Overview

The Import Menu contains two main sections:

  • Locations
  • Employees

Each section contains two lists:

  • Import List — Items that will be imported.
  • Skip List — Items that will not be imported.

All locations and employees found in the selected file will initially appear in the Import List.

Selecting What to Import

In between each list are arrow buttons:

  • Right Arrow — Moves selected items to the Skip List.
  • Left Arrow — Moves selected items back to the Import List.

Any items placed in the Skip List will not be imported.

This allows you to selectively import only the data you desire.

Additional Import Options

Below the lists are two optional checkboxes that are on by default:

  • Import Settings
    • When checked, the selected file's settings will replace the current sheet's settings.
  • Import District Manager
    • When checked, the District Manager and their schedule will also be imported from the selected sheet.

Completing the Import

After making your selections:

  1. Click the Import button.

The following data will be imported:

  • All selected locations.
  • All selected employees.
  • Selected employee's default schedules.
  • Selected employee's current schedules.

Important Notes and Behavior

  • If an employee is scheduled at a location that is not imported and does not already exist in the current sheet:
    • That employee will be set to Off for those days.
    • Their default schedule will also reflect this change.
  • After the import is completed, a

    popup message

    will appear listing any employees affected by missing locations.

Duplicate Name Restrictions

If the current sheet already contains a location name or an employee name that matches any employee or location in the selected file, that item will not be available for import.

If you would like to import that item you must first delete or rename the existing location or employee in either sheet so that they do not match, then run the import again.

The Load Previous Schedule feature allows you to load a prior active schedule for selected employees from another file.

This is useful when you want to:

  • Reuse a past week's schedule.
  • Only select employees have alternating weekly schedules.
  • Copy schedules without affecting locations or default settings.

Opening the Load Menu

  1. In the top-left corner of the Excel sheet, click the Additional Options button.
  2. An options menu will appear.
  3. Select Load Previous Schedule.

    4. This will open the file explorer.

  4. Select another Schedule Maker Excel File.

    5. After selecting a compatible file, the Load Menu will appear.

Load Menu Overview

This menu contains a single section for Employees. Two lists are shown:,/p>

  • Import List — Employees whose schedules will be loaded.
  • Skip List — Employees whose schedules will not be loaded.

By default, all employees that exist in both the current sheet and the selected file will appear in the Skip List.

Selecting Employees to Load

In between the lists are arrow buttons:

  • Left Arrow — Moves selected employees to the Import List.
  • Right Arrow — Moves selected employees back to the Skip List.

Any employees placed in the Skip List will not have their schedules changed and loaded.

Completing the Schedule Load

Once your selections are made, click the Import button to load the previous schedule.

Important Notes and Behavior

  • This feature only imports active schedules.
    • This feature does not import locations, employee default schedules, settings.
  • Only employees that exist in both the current sheet and the selected file will be available for selection.
    • If an employee exists in your current sheet but is not found in the selected file, you wil not have the option to load a schedule for them.
  • If you load in an employee's schedule, but do not have the same locations in the current sheet, the employee will be schedules Off on those days.
